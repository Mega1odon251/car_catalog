# Car Catalog
## Аналитические запросы

В файле `sql/queries.sql` реализованы следующие аналитические запросы:

1. **Топ-3 самых дорогих автомобилей** — сортировка по убыванию цены
2. **Средняя цена по марке** — группировка с сортировкой от дорогих к дешёвым
3. **Фильтр автомобилей** — новее 2021 года и дешевле 3 млн рублей
4. **Распределение по цветам** — только цвета, представленные минимум 2 автомобилями
5. **Классификация по возрасту** — "новый" (≤ 3 лет) или "старый"

Все запросы используют реальные данные из таблицы `cars`.
